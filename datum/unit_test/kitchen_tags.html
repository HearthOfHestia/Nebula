kitchen_tags
/datum/unit_test/kitchen_tags

Each recipe's fruit entries should correspond to some plant's kitchen_tag.

Seed datums define kitchen_tags and traits that determine if they can be modified in certain ways, while recipes define combinations of tags and traits that they need to succeed. This test ensures that all entries in the latter are valid.
